# iNotebook - MERN Stack Notes App

**iNotebook** is a full-stack MERN application for managing personal notes. It includes user authentication (Signup/Login), note CRUD operations, dynamic search, and a beautiful Bootstrap-based responsive UI.

---

## 🌟 Features

- **User Authentication**  
  - Signup / Login with email and password  
  - JWT-based authentication with secure password hashing (bcrypt)  
  - "Remember Me" feature for persistent login  

- **Notes Management (CRUD)**  
  - Create, Read, Update, Delete notes  
  - Each note has title, description, and optional tags  

- **Search Notes**  
  - Real-time search on note titles and descriptions  

- **Frontend UI**  
  - Responsive and interactive design using **Bootstrap 5**  
  - Gradient cards, shadow effects, and alert notifications  

- **Backend API**  
  - Built with **Node.js** + **Express.js**  
  - Connected to **MongoDB Atlas** database  
  - Secure and structured routes (`/api/auth`, `/api/notes`)  
  - CORS enabled for frontend-backend communication
